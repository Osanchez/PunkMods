# PUNK Modding Docs

Reverse-engineering documentation for modding **PUNK** (Unity 6.3, Mono backend), generated
from a decompile of `Punk_Data/Managed/Punk.Main.dll`.

👉 **Start with [`00-overview.md`](00-overview.md)** — moddability assessment, toolchain
setup (BepInEx + Harmony), a plugin skeleton, and the full index of subsystem docs.

The docs are numbered by subsystem (`01`–`15`). Read `01-core-architecture.md` next; it
explains the dependency-injection bootstrap and the cleanest places to hook a plugin.
